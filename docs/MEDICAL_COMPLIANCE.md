# 의료 법규 준수 가이드

## 1. 개요

성형외과 웹사이트는 의료법, 의료광고 심의기준, 개인정보보호법 등 관련 법규를 준수해야 합니다.

---

## 2. 의료광고 심의기준

### 2.1 의료광고 사전 심의

**대상:** 의료기관이 직접 운영하는 웹사이트

**심의 기관:** 보건복지부 지정 의료광고 심의기관

**필수 사항:**
- 의료광고 사전 심의 신청
- 심의 필증 홈페이지 표기

### 2.2 금지 표현

| 구분 | 금지 표현 | 허용 표현 |
|-----|----------|----------|
| 시술 효과 | "완벽한 결과", "100% 성공" | "자연스러운 결과", "높은 만족도" |
| 기간 | "영구적", "평생" | "장기 지속", "오랫동안" |
| 비교 | "국내 최고", "1위" | "전문 의료진", "풍부한 경력" |
| 보장 | "부작용 없음", "무조건 환불" | "안전한 시술", "사후 관리" |
| 의미 불명 | 외국어, 약어 등 | 알기 쉬운 우리말 |

### 2.3 필수 표기 사항

**의료광고 심의 필증:**
```html
<div class="ad-approval">
  <p>의료광고 심의필증 제 <strong>2024-0000호</strong>호</p>
  <p>심의일: 2024년 3월 1일</p>
  <p>유효기간: 2027년 2월 28일</p>
</div>
```

**시술 전후 사진 주의사항:**
```html
<div class="before-after-notice">
  <p>* 사진은 실제 시술 환자의 결과물입니다</p>
  <p>* 개인마다 시술 결과는 다를 수 있습니다</p>
  <p>* 모델의 허락을 받아 게시하였습니다</p>
</div>
```

## 3. 의료법 준수사항

### 3.1 의료기관 정보 표기

**필수 표기:**
- 의료기관명
- 의료기관 주소
- 대표자명 (의사 성명)
- 의료기관 전화번호
- 사업자등록번호
- 의료기관 종별

```html
<section class="hospital-info">
  <h2>OO성형외과의원</h2>
  <dl>
    <dt>주소</dt><dd>서울특별시 강남구 테헤란로 123</dd>
    <dt>대표의사</dt><dd>김OO</dd>
    <dt>전화번호</dt><dd>02-1234-5678</dd>
    <dt>사업자등록번호</dt><dd>123-45-67890</dd>
    <dt>개설일</dt><dd>2020년 3월 1일</dd>
  </dl>
</section>
```

### 3.2 의료진 정보

**필수 표기:**
- 의사 성명
- 면허 종류 (의사)
- 면허 번호
- 전문과목
- 소속 병원

```html
<div class="doctor-info">
  <h3>김OO 원장</h3>
  <ul>
    <li>의사면허번호: 제12345호</li>
    <li>전문과목: 성형외과</li>
    <li>전문의 자격번호: 성형전문의 제678호</li>
    <li>경력: OO대학교병원 성형외과 전임의 등</li>
  </ul>
</div>
```

### 3.3 시술 정보

**필수 포함:**
- 시술 명칭
- 시술 내용
- 주요 부작용
- 주의사항

```html
<div class="procedure-info">
  <h3>쌍꺼풀 수술</h3>
  <p><strong>시술 내용:</strong> 눈 위쪽 피부를 절개하여 쌍꺼풀을 만드는 시술입니다</p>

  <div class="side-effects">
    <h4>주요 부작용</h4>
    <ul>
      <li>일시적인 붓기, 멍</li>
      <li>감염, 출혈</li>
      <li>흉터</li>
    </ul>
  </div>

  <div class="cautions">
    <h4>주의사항</h4>
    <ul>
      <li>시술 후 일주일간 안경 착용 권장</li>
      <li>술, 담배 일주일 금지</li>
      <li>뜨거운 사우나 일주일 금지</li>
    </ul>
  </div>
</div>
```

## 4. 개인정보보호법

### 4.1 개인정보 수집 동의

**회원가입 시 필수 동의:**
- 개인정보 수집/이용 동의
- 제3자 제공 동의 (해당 시)
- 민감 정보 처리 동의 (건강 정보)

```html
<form class="signup-form">
  <div class="terms">
    <label>
      <input type="checkbox" name="privacy" required />
      <span>개인정보 수집/이용에 동의합니다 (필수)</span>
    </label>

    <button type="button" class="view-terms">전문보기</button>

    <!-- 전문 -->
    <dialog class="terms-dialog">
      <h3>개인정보 수집/이용 동의서</h3>
      <pre>
1. 수집 항목: 이름, 이메일, 비밀번호, 연락처
2. 수집 목적: 회원 관리, 상담 서비스 제공
3. 보유 기간: 회원 탈퇴 시 즉시 파기
      </pre>
    </dialog>
  </div>
</form>
```

### 4.2 개인정보처리방침

**필수 항목:**
1. 개인정보의 처리 목적
2. 개인정보의 처리 및 보유 기간
3. 개인정보의 제3자 제공
4. 개인정보의 처리 위탁
5. 정보주체의 권리/의무 및 행사 방법
6. 개인정보의 파기 절차/방법
7. 개인정보의 안전성 확보 조치
8. 개인정보 보호책임자
9. 영상정보처리기기 운영/관리 (해당 시)

### 4.3 건강정보 (민감 정보)

**특별 주의:**
- 건강정보는 민감 정보로 분류
- 별도 동의 필요
- 보안 강화 요구

```html
<label class="health-info-consent">
  <input type="checkbox" name="healthInfo" required />
  <span>
    상담 과정에서 수집되는 건강정보(시술 문의 사항 등)에 대한
    수집/이용에 동의합니다 (필수)
  </span>
</label>
```

## 5. 콘텐츠 가이드라인

### 5.1 블로그 포스트

**주의사항:**
- 의료 정보는 정확하고 객관적이어야 함
- 개인적인 의견이 아닌 의학적 근거 제시
- 미래의 환자에게 맹목적인 기대 심어주지 않기

**작성 가이드:**
```markdown
# 쌍꺼풀 수술 가이드

## 개요
쌍꺼풀 수술은 가장 많이 시행되는 성형 시술 중 하나입니다.

## 시술 방법
- 절개법: ...
- 묶음법: ...

## 예상 결과
...
* 개인마다 결과는 다를 수 있습니다

## 부작용
...

## 주의사항
...
```

### 5.2 상담 게시판

**관리자 답변 가이드:**
- 의학적 사실에 기반한 답변
- 과장된 효과 약속 금지
- 직접 시술해야 함을 명시

**예시 답변:**
```
안녕하세요, OO성형외과입니다.

문의하신 쌍꺼풀 수술에 대해 답변드립니다.

1. 시술 방법: ...
2. 예상 비용: ...
3. 회복 기간: ...

정확한 진단을 위해 병원 방문 상담을 권장합니다.
면담 후 개인에 맞는 수술 계획을 세워드립니다.

감사합니다.
```

**필수 사항:**
- GDPR 준수 동의
- 데이터 이전 권리
- 잊힐 권리

```html

  <p>
    If you are a resident of the European Union, you have additional rights
    regarding your personal data. Please refer to our Privacy Policy.
  </p>
</div>
```

### 6.2 중국인 대상

**PIPL (중국 개인정보보호법):**
- 중국인의 개인정보는 국내에 저장
- 별도의 동의 절차
