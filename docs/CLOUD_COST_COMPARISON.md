
## 9. 호출 빈도가 낮을 때의 이점

### 9.1 Cloudflare 무료 플랜

- **100,000회/일** 요청 가능
- 월 5,000회 사용 시 → **0.5% 활용**
- 여유: 99.5%

### 9.2 서버 사양 최적화

- 호출이 적으면 사양 낮춰도 됨
- t2.micro / nano 사양으로 충분
- 비용 50~70% 절감
